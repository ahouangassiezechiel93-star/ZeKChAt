# ZeKChAt
ZeKChAt est une app de messagerie instantanée en un seul fichier HTML, propulsée par Supabase. Elle offre la messagerie privée &amp; groupes en temps réel, les statuts 24h, le typing indicator, la présence en ligne, le partage de fichiers, les messages vocaux, les réactions et les appels audio/vidéo via WebRTC. Stack : HTML ·CSS · JS vanilla · Supabase
